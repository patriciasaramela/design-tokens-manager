# Gerenciamento de Design Tokens

Este projeto é uma aplicação React desenvolvida para gerenciar design tokens de um design system corporativo multibrand. 
Ele oferece uma solução completa para documentar, visualizar, criar e distribuir tokens de design, promovendo colaboração eficiente entre equipes de design e desenvolvimento.
O projeto do Desin System tem como objetivo unificar o desenvolvimento de um unico design system, para múltiplas marcas, e o gerenciador de tokens possibilita a visualização rápida e fácil de estilos específicos para cada identidade corporativa.

## Principais funcionalidades

**Cadastro de novos tokens**:

Permite que designers registrem novos tokens diretamente por uma interface.
Os tokens cadastrados podem ser testados em tempo real para verificar sua aplicação e funcionamento no CSS.

**Documentação e visualização**:

Exibição da aplicaçãp dos design tokens, organizados por categorias (como cores, tipografia, espaçamento, etc.).
Interface intuitiva para facilitar a navegação e o entendimento do design system.

**Exportação de Tokens**:

Opção para baixar os tokens no formato necessário, garantindo o correto funcionamento.

**API**:
Integração com API do Airtable para armazenar e manter os tokens sincronizados.
(Anteriormente usava uma planilha google integrada por API para o funcionamento)

## Como executar

Para que o projeto funcione corretamente é preciso inserir os dados no arquivo .env
Caso não tenha acesso aos dados, entrar em contato com o time de Design de Produtos.
